# Semana 1 - Base comunicativa para Data Analyst

Meta: presentarte, describir tu enfoque y pedir claridad sin perder confianza.

---

## Dia 1 - Quien soy (15-20 min)

### Decir esto

1. Hi, I'm [name]. Nice to meet you.
2. I'm learning data analysis and practical SQL.
3. I'm improving my English for analytics work.
4. I'm a beginner, but I practice every day.
5. Could you repeat that a bit more slowly?

### Mini regla

Present simple para rutina: I study / I practice / I work.

### Practica

Completa en voz alta:
- My name is ___.
- I'm learning ___.
- I want to improve my English because ___.

---

## Dia 2 - Mi rutina de estudio (15-20 min)

### Decir esto

1. I study data for about [time] every day.
2. I review SQL queries and dashboard examples.
3. I take notes about new terms.
4. I practice speaking after each lesson.
5. My goal is to explain insights clearly.

### Palabras del dia

| English | Espanol |
|---------|---------|
| dataset | conjunto de datos |
| query | consulta |
| insight | hallazgo |
| trend | tendencia |

---

## Dia 3 - Pedir contexto (15-20 min)

### Decir esto

1. Can you clarify the metric?
2. What time range should I use?
3. Do you want totals or averages?
4. Should I segment by country or product?
5. I want to confirm before I query the data.

### Practica

Simula una pregunta a un stakeholder en 3 frases.

---

## Dia 4 - Explicar una tabla simple (15-20 min)

### Decir esto

1. This table has one row per transaction.
2. The primary key is transaction_id.
3. The column amount is in USD.
4. There are some null values in customer_id.
5. We should clean this before analysis.

### Practica

Explica una tabla en 5 frases: que es, llave, columna clave, problema, accion.

---

## Dia 5 - Opinion profesional (15-20 min)

### Decir esto

1. In my opinion, this metric is useful.
2. I think we need more context.
3. This chart is clear, but we can improve labels.
4. The result looks good, but we should validate it.
5. Consistency matters more than speed.

### Practica

Da 3 opiniones sobre un reporte: una fuerte, una duda, una mejora.
