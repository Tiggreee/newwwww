# Data English Sprint - Interactive route (A2 to B1)

Goal: help beginners speak and write in English while learning the full data route from zero to practical SQL.

Promise: short, persuasive, simple, and challenging.

## Is the current material reliable?

Short answer: yes, now aligned to data route with verified references.

- Strong points: practical speaking flow, progressive weekly structure, interview language for analytics.
- Scope: foundations, design, SQL, and communication of insights.

## Route map (from less to more)

### Level 1 - Database foundations (Week 1)

What you learn:
- What a table, row, column, key, and null are.
- Why constraints protect data quality.
- How to read simple SELECT queries.

English output challenge:
- Explain a table in 5 lines: purpose, fields, key, one example row, one risk.

### Level 2 - Database design (Week 2)

What you learn:
- Entities, relationships (1:1, 1:N, M:N).
- Junction tables for many-to-many.
- Basic normalization (1NF to 3NF).

English output challenge:
- Present your ER model in 60 seconds: entities, keys, relationship rules.

### Level 3 - Programming with SQL (Week 3)

What you learn:
- SELECT, WHERE, ORDER BY, GROUP BY, HAVING.
- JOIN types and when to use each.
- INSERT, UPDATE, DELETE and safe habits.

English output challenge:
- Explain one query in 6 lines: objective, filter, join rule, grouping rule, expected output, edge case.

### Level 4 - Storytelling and interviews (Week 4)

What you learn:
- Explain findings to technical and non-technical audiences.
- Handle interview questions with confidence.
- Close with recommendations and next steps.

English output challenge:
- Present one insight in 60-90 seconds with number + context + recommendation.

## Interactive flow

1. Open Data Route Interactive in the website.
2. Review one visual card.
3. Answer one challenge question.
4. Repeat the speak prompt out loud.
5. Log your weak phrase in bitacora.md.

## Persuasion script for landing

You do not need perfect English to start in data. You need a clear path, daily output, and real SQL practice. This route takes you from table basics to insight communication in short focused steps.

## Stretch challenge

- Which cohort is at risk of dropping?
- Which module has the highest average score?
- Which learners improved week over week?

Then explain your logic in plain English in under 90 seconds.
